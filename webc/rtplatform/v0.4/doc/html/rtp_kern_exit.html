void rtp_kern_exit

( int exitvalue )

Exit an application.

Documentation

Exit an application. The exit value is 0 if successfully exiting, and -1 if exiting with a failure.

Parameters:
exitvalue - Exit value; 0 for success, -1 otherwise.
Returns:
void

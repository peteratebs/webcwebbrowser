void _rtp_debug_print_mem_usage

(const char* logFile,
  const char* mode)

Identify any memory corruption.

Documentation

Identify any memory corruptions. If RTP_MEM_DEBUG_FILE this prints the information to a file indicated by the logFile name passed in. If the logFile is NULL and RTP_MEM_DEBUG_PRINTF is defined, the information will be sent to rtp_printf.

Returns:
void
Preconditions:
Only use if dynamic memory is required. MUST not be called directly. Use the rtp_debug_PrintMemLeaks macro defined in rtpmem.h and turn on RTP_TRACK_LOCAL_MEMORY. The rtpstdio file system support and/or the rtpterm terminal layer must be present to make use of this debugging feature.

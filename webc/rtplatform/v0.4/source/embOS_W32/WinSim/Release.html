Tool chain used for build

The following tools have been used:
Compiler:  Visual C++ V6
Workbench: Visual Studio V6
Note that embOS Simulation is written in ANSI-"C" and uses Windows 32bit API.
It does not rely on Microsoft Visual studio and may also be used with other tool chains.

New features

Version 3.20

  1. OS_Suspend() / OS_Resume() implemented

    2. New functions to suspend and resume tasks unconditionally.
  2. Dynamic memory allocation supported by embOS

    3. Thread safe dynamic memory allocation functions implemented.

Improvements

Version 3.20

    NONE, initial release of embOS Simulation

Program corrections

Version 3.20c

  1. OS_GetMailTimed() corrected
    OS_GetMailTimed () did not retrieve a mail after 256 calls, the waiting task remained suspended. Corrected with version 3.20c of embOS.
    This problem exists in all previous embOS versions and is fixed with version 3.20c.

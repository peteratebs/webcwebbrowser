int webc_ElementSetChecked

(HELEMENT_HANDLE element,
  HTML_BOOL checked,
  HTML_BOOL refresh)

Set checked status for checkbox or radio button

Documentation

If the element handle passed to this routine is that of an HTML_CHECKBOX_ELEMENT, or an HTML_RADIO_BUTTON_ELEMENT, then the 'checked' status of the element will be set according to the boolean 'checked' parameter. If this function is called with a 'checked' value of WEBC_TRUE, then subsequent calls to webc_ElementGetChecked will return the string set as the value for this element, via webc_ElementSetValue. If WEBC_FALSE is passed to this routine, then subsequent calls to webc_ElementGetChecked will return WEBC_NULL.
Parameters:
element - The element to set
checked - Whether or not the element should be checked
refresh - If WEBC_TRUE, then redraw this element
See Also:
HELEMENT_HANDLE, CHECKED, webc_ElementGetChecked, webc_ElementSetValue, webc_ElementRefresh

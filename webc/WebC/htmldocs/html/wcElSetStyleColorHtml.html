void wcElSetStyleColorHtml

(wcEL element, CSSPropertyType Prop,
  HTMLColor color, wcBOOL refresh)

Set the color of an element using CSS.

Documentation

Formats a string and passes it to wcElSetStyle() to set the color an element using CSS. Uses the same color representation that is returned from wcElGetColorValue().

Parameters:
wcEL - Element - The element.
Prop - Which color property to set.
HTMLColor - Color - The type returned from wcElGetColorValue().
refresh - Redraw the element immediately if wcTRUE.
Returns:
nothing.
See Also:
wcElSetStylePos,,wcElSetStyleColorRgb, wcElSetStyleColorNamed

HTMLColor wcImageGraphNameToColor

(HtmlColorIndex ColorName)

Convert a color index value to a 32 bit html value that may be used as a color argument to wcImageGraphSetPenColor.

Documentation

Convert color names to 32 bit rgb values. The names are the same as those passed in strings to wcElSetStyleColorNamed except theey are integer constants instead of strings. 



 Example:
	wcImageGraphSetPenColor(ImageElement, wcImageGraphNameToColor(DarkSlateBlue)); // Set the pen to DarkSlateBlue

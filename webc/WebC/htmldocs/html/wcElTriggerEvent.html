HTMLEventStatus wcElTriggerEvent

(wcEL El, HTMLEvent* pEvent,
  int delay)

Send an event to an element immediately or with delay.

Documentation

Pass an event to an element's event handler routine.

Parameters:
wcEL - Element - The element.
HTMLEvent - * pEvent - Initialized event structure to pass to the handler.
int - delay - Delay in miliseconds before triggering the event. Zero for no delay.
Returns:
HTMLEventStatus. If Delay is non-zero HTML_EVENT_STATUS_CONTINUE otherwise the return value from the element's event handler.

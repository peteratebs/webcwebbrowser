wcBOOL wcElGetIdMatch

(wcEL element, char* string)

Test if an element's Id attribute matches an ascii string.

Documentation

Returns wcTRUE if the Id matches the string. Helpful when executing wcDocEnumerate to find elements with specific Id's.

Parameters:
wcEL - Element - The element.
wcEL - char * - The string.
Returns:
char *Id wcTRUE if they match.

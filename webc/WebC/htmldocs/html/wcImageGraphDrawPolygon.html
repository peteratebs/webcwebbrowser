void wcImageGraphDrawPolygon

(wcEL ImageElement, int nPoints,
  int* pointArray, wcBOOL fill)

Draw a sequence of lines to an image element background canvas.

Documentation

Draws a sequence of lines at the current pen width and color onto the background canvas element at coordinates provided by the caller in an array.

Parameters:
wcEL - ImageElement - An image element that is the background canvas for our draw operations.
int - nPoints - Number of x,y coordinate pairs stored at the location in the pointArray argument.
int - *pointArray - A pointer to an array of (nPoints*2) integers containing the x,y offsets in the image array of a connected set of lines to draw.
wcBOOL - fill - Fill the area bounded by pointArray if this argument is wcTRUE.
Returns:
nothing

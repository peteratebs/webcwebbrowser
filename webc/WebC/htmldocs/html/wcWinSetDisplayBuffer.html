wcBOOL wcWinSetDisplayBuffer

(wcCtx* Ctx, unsigned char* buffer)

Set a buffer to draw to for the window. Graphics will not be written to the framebuffer hardware but will instead be directed this buffer.

Documentation

Set a buffer to draw to for the window. Graphics will not be written to the framebuffer hardware but will instead be directed this buffer

Parameters:
wcCtx - * Ctx - Window to change the display buffer on.
unsigned - char *buffer - New buffer to write to, if zero a new buffer is allocated internally.
Returns:
wcTRUE if the operation was succesful wcFALSE otherwise.

wcWinUnSetDisplayBuffer

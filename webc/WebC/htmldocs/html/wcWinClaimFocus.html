void wcWinClaimFocus

(wcCtx* Ctx)

Claim focus for a browser window.

Documentation

Instructs webC to disard all events not destined for the current window. The Claim may be released by calling wcWinReleaseFocus. The claim is automatically released if wcWinClose is called. wcWinClaimFocus is called by wcWinCreate and wcWinCreateEx when a Modal window is requested.

Parameters:
wcCtx - *Ctx Context to claim focus for.
Returns:
Nothing
See Also:
wcWinReleaseFocus

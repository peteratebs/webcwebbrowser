#define WEBC_CFG_EVENT_QUEUE_SIZE

Configure the size of HTMLEvent queue.
This is the maximum number of Events that webC will queue. If the maximum is exceeded new events are dropped. The default value is low, raise it if you suspect events like mouse move or touchscreen tracking events are being dropped. In debug mode an assert is generated if the queue fills.

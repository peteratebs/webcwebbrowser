int webc_ElementSetSrcASCII

(HELEMENT_HANDLE element, char* _src,
  HTML_BOOL refresh)

Set the source url for an image or frame element

Documentation

Sets the src attribute for the specified element. Only affects HTML_FRAME_ELEMENT and HTML_IMAGE_ELEMENT. Doesn't load or redraw (unless 'refresh' is set to WEBC_TRUE) until webc_ElementRefresh is called.
Parameters:
element - The element to set
src - The new src
refresh - If WEBC_TRUE, then load the new url and redraw this element
See Also:
HELEMENT_HANDLE, SRC, webc_ElementGetSrc, webc_ElementRefresh

Example Source Code

Source code fragments that use webC apis to solve common user interface problems. The source code for multiple examples is provided in webcapiexample.c. Click on links on this page to review individual examples. Note: You may cut and paste fragments from the documentation provided but you should consult webcapiexample.c for the latest source code.
OpenExampleChildWindow
Responsible for opening the window context used by example below.
ExDocEventHandling
This example demonstrates using document event handlers.
ExElEventHandling
This example demonstrates using element event handlers.
ExDragButtons
This example demonstrates using dynamic HTML from 'C' create a pallet of buttons that may be dragged and inserted into a target area.
ExCalculator
This example demonstrates mixing html and css source with 'C' source code to implement a simple functional calculator in 40 lines of CSS and HTML and < 100 lines of C.
  • Demonstrate using wcDocEnumerate to access elements in an html document.
ExCalculatorDynamic
This example demonstrates wcTableApi functions to implement a calculator function similar to ExCalculator using dynamically generated content instead of CSS and HTML.
  • Demonstrate using wcNewStyledTable wcTableInsertRow and wcTableInsertCell to dynamically create a table.
ExPresentationSlideShow
Using pure dynamic HTML from 'C' create elements and display them using varying style changes.
ExDemonstrateWebcVirtualFiles
This example registers precompiled content with the read only file system layer and loads content from it.
ExResizeBox
This example demonstrates a rectangular area that can be resized and dragged.
ExSoftKeyPad
This example demonstrates initializing and using the built in soft keypad.

wcElPopEventHandler

wcElTriggerEvent

wcDocSetEventHandler
Assign an event handler to the body element of this document.
wcDocPopEventHandler
Restore the previous event handler for this document's body element.
wcDocTriggerEvent
Send an event to the document's body element either immediately or with a delay.

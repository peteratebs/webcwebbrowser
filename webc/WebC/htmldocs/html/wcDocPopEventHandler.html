void wcDocPopEventHandler

(wcCtx* Ctx)

Restore the previous event handler for this document's body element.

Documentation

Pop the previous event handler for the body element from an internal stack and make it the new event handler for the specified element. This function returns the event handler to the handler prior to the last call to wcDocSetEventHandler.

Parameters:
wcCtx - * Ctx - The current wc context. 
Example:   See the example for wcDocSetEventHandler.

Returns:
Nothing
See Also:
wcDocSetEventHandler

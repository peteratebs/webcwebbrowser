void wcImageGraphSetPenColor

(wcEL ImageElement,
  HTMLColor color)

Set the color for subsequent draw operations.

Documentation

Sets the 32 bit color for subsequent draw operations. 

 The bit positions in the color parameter are:
 0-7	Blue
 8-15	Green
 15-23  Red
 24-31	Alpha
 note: The current release does not support Alpah channel for wcImageGraph operation.

Parameters:
wcEL - ImageElement - An image element that is the background canvas for our draw operations.
HTMLColor - color - The bit color ARGB color value to draw in. You may also used named colors by calling wcImageGraphNameToColor.
Returns:
nothing

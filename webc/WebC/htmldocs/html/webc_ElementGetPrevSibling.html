HELEMENT_HANDLE webc_ElementGetPrevSibling

(HELEMENT_HANDLE element)

Get the prev sibling of an element

Documentation

Returns the prev sibling element of the given element.

Note: The wcEl support library provides a simplified DOM traversal package. Use of that interface is recommended for new projects.
Parameters:
element - The element whose sibling to get
Returns:
The handle of the prev sibling element or WEBC_NULL if this is the first child.

int wcRegisterMetaCallback

(const char* pMetaName,
  HTMLMetaCallbackFn Callback)

Register a user supplied Metatag handler.

Documentation

Register a 'C' callback routine to be called when the named metatag is encountered in an html header.

Metatags callbacks are an easy way to provide users with access to 'C' native functions from their callable from Load optional initial content and sets a user supplied handler function to be called when the view is created. The Window config structure can be used to set the border color, background color, and border style. The Window config structure can be used to control window properties including if it is modal and if it is resizable.

Parameters:
const - char *pMetaName Name of the metatag to process when encountered in an html header.
HTMLMetaCallbackFn - Callback Function to call when the metatag is encounetered. 
The callback function must be of the form:
HTMLMetaStatus *CallbackFn(const char* pContent, wcCtx *pCtx)
where:
 pContent is the value from the content= field in the document.
 pCtxt is an initialize webC application context.
The callback must return one of these values:
    HTML_META_STATUS_CONTINUE - Instructs webC to continue processing the page including processing any additional metatags
    in the header or to continue to the body.
    HTML_META_STATUS_STOP - Behaves the same as HTML_META_STATUS_CONTINUE.
    HTML_META_STATUS_HALT - Stops loading the page immediately.

Returns:
rwcRegisterMetaCallback returns 0 if it was able to register the calback, -1 otherwise.

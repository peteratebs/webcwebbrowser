Pop open a keypad.

Documentation

Opens a keypad control. This routine is called when text edit control receives a mouse click and WEBC_SUPPORT_TOUCHSCREEN is enabled. This function is not explicitly part of the API but it is listed here in case the need for a wcPoPUpKeyPad() API entry arrises. Study the source code for usage examples.

Returns:
Nothing

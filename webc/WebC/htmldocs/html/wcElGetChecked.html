wcBOOL wcElGetChecked

(wcEL element)

Get the status of a checkbox or radio button

Documentation

Return wcTRUE if the given element handle is a radio button element and selectyed or a checkbox element and it is checked.
Parameters:
wcEL - element - The element to test the checked attribute for.
Returns:
The wcTRUE if the element is checked wcFALSE if it is not checked
See Also:
wcElSetChecked, wcElSetValue

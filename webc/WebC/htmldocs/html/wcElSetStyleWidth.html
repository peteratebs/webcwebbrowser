void wcElSetStyleWidth

(wcEL element, int width, wcBOOL refresh)

Set the width of an element using CSS.

Documentation

Formats a string and passes it to wcElSetStyle() to set the width using CSS.

Parameters:
wcEL - Element - The element.
int - width - New width.
refresh - Redraw the element immediately if wcTRUE.
Returns:
nothing.
See Also:
wcElSetStylePos,wcElSetStyleRgb,,wcElSetStyleHtmlColor

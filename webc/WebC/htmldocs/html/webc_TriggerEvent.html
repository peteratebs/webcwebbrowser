HTMLEventStatus webc_TriggerEvent

(HELEMENT_HANDLE element,
  HTMLEvent* pEvent)

Manually trigger an HTMLEvent

Documentation

This method can be used to manually trigger event handling for a particular event on a specific element.

Parameters:
element - Handle of the element to receive the event
pEvent - Pointer to already-initialized HTMLEvent structure

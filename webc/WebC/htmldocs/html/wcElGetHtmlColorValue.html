Documentation

Get a property of type color from Html

Note: To know how what value will be used when an element is displayed, First consult wcElGetCssColorValue and if that fails (returns -1) then consult wcElGetHtmlColorValue.
Parameters:
element - The element whose color to get
Prop - Which property to retrieve
color - Pointer to an HTMLColor (unsigned long) variable to fill in with the color information for this element.
Returns:
0 if the property was found and color was filled with a valid color, -1 otherwise

void wcElSetBgImage

(wcEL element, char* _bgimg, wcBOOL refresh)

Set an element's background image url

Documentation

Sets the background image url for the specified element. Only affects HTML_BODY_ELEMENT, HTML_TABLE_ELEMENT, HTML_TABLE_CELL_ELEMENT, and HTML_TABLE_ROW_ELEMENT. Doesn't load the new bitmap or redraw (unless 'refresh' is set to WEBC_TRUE) until webc_ElementRefresh is called.
Parameters:
element - The element to set
bgimg - The new background image url
refresh - If wcTRUE, then load the new url and redraw this element

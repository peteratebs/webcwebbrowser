The webc: url (also accessible as rom:) is a rom based read only file system that enables you to include content in your application like you do from a file system.

Documentation

The rom file system lets you prototype content with webC on a PC workstation, organizing it using typical names like "images", "metadata" and "html". You can then burn this content it into your application using makewebcfs and acess it using the rom: url or the webc: url.

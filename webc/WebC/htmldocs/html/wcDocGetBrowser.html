wcBROW wcDocGetBrowser

(wcDOC doc)

Retrieve an HTML Document's parent HTML Browser

Documentation

Parameters:
doc - The handle of the document 
Example - Initialize a context from an element handle.
wcCtxtInit(wcDocGetBrowser(wcElGetDocument(element)), wcElGetDocument(element));

Returns:
The handle of the HTML Browser containing the given document.

char* webc_ElementGetValueASCII

(HELEMENT_HANDLE element)

Get the value of an input widget

Documentation

If the given element handle is an HTML_EDIT_STR_ELEMENT (text field), HTML_EDITBOX_ELEMENT (text box), or HTML_SELECT_ELEMENT (combo box), then this function will return the currently entered value of the widget. For element types that have no input value (everything that is not a widget), this function will return WEBC_NULL. To retrieve the status of boolean type input widgets (such as radio buttons and checkboxes), use webc_ElementGetChecked.
Parameters:
element - The element whose value to get
Returns:
The element's value or WEBC_NULL if it doesn't have one.
See Also:
HELEMENT_HANDLE, VALUE, webc_ElementSetValue, webc_ElementGetChecked

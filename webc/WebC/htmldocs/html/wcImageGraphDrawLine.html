void wcImageGraphDrawLine

(wcEL ImageElement, int x0, int y0,
  int x1, int y1)

Draw a line to an image element background canvas.

Documentation

Draws a line at the current pen width and color onto the background canvas element at the requested coordinates.

Parameters:
wcEL - ImageElement - An image element that is the background canvas for our draw operations.
int - x0 - Starting point x,y coordinate.
int - y0
int - x1 - End point x,y coordinate.
int - y1
Returns:
nothing

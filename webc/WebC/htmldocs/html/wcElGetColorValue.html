int wcElGetColorValue

(wcEL element, CSSPropertyType Prop,
  HTMLColor* color)

Get a property of type color from Css or html.

Documentation

Get a property of type color from Css. First checks Css, if not found consults html for default values. Note: The property must ve one that is expressed in color otherwise the result will be meaningless. 
		These are the most common properties that applications code might query.


		CSS_PROPERTY_COLOR
		CSS_PROPERTY_BACKGROUND_COLOR


		These properties may also be queried.


		CSS_PROPERTY_ALINK_COLOR
		CSS_PROPERTY_VLINK_COLOR
		CSS_PROPERTY_LINK_COLOR
		CSS_PROPERTY_OUTLINE_COLOR


		CSS_PROPERTY_BORDER_BOTTOM_COLOR
		CSS_PROPERTY_BORDER_COLOR
		CSS_PROPERTY_BORDER_LEFT_COLOR
		CSS_PROPERTY_BORDER_RIGHT_COLOR
		CSS_PROPERTY_BORDER_TOP_COLOR
Parameters:
element - The element whose color value to get
Prop - Which property to retrieve
color - Pointer to an HTMLColor (unsigned long) variable to fill in with the color information for this element.
Returns:
0 if the property was found and color was filled with a valid color, -1 otherwise

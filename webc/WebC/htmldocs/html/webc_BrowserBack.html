int webc_BrowserBack

(HBROWSER_HANDLE browser)

Load the previous location in the history

Documentation

This function is functionally identical to pressing the "Back" button in an HTML Browser window and may be used to traverse hierarchical page structures.

Parameters:
browser - The handle of the HTML Browser
Returns:
0 on success, -1 otherwise
See Also:
webc_BrowserForward, webc_BrowserHome, webc_BrowserRefresh, webc_BrowserLoadUrl

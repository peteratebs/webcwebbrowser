void wcSoftKeyDisable

(void)

Disable webc's soft keyboard.

Documentation

This function is typically not used for touchscreen based applications. It is used for demonstrating the touchscreen support on desktop systems where wcSoftKeyDisable wcSoftKeyEnable enables the keyboard for a demonstration (ExSoftKeyPad)and then disables it when the example applet finishes.

Returns:
Nothing.

void wcWinClose

(wcCtx* Ctx)

Close a browser window.

Documentation

Close the current browser view, call object destroy message handlers and release internal resources.

Parameters:
wcCtx - *Ctx Context of the window to close.
Returns:
Nothing.
See Also:
wcWinCreateEx, wcWinCreate

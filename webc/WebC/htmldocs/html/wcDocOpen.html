int wcDocOpen

(wcCtx* Ctx)

Open an HTML Document for writing

Documentation

This function must preceed any calls to wcDocWriteHtml and wcDocWriteString

Parameters:
Ctx - The current wc context.
Returns:
0 if the operation was successful, -1 otherwise.
See Also:
HDOC_HANDLE, wcDocClear, wcDocClose, wcDocWriteHtml, wcDocWriteString

wcEL wcNewStyledTextArea

(wcCtx* Ctx, wcEL ParentEl,
  wcBOOL insert, int top, int left,
  int height, int width, int zindex,
  char* color, char* bgcolor,
  char* bgimage, char* style)

Constructor function for a textarea element with common styles applied to it.

Documentation

Creates a textarea element and customizes it with CSS based on the the values original in the function arguments. If requested the new element is also inserted into the document. If you can't conveniently completely style the element your needs with this constructor function you can modify the returned element handle by calling wcElSetStyle or one of its derivatives.

Parameters:
wcCtx - * Ctx The current wc context.
wcEL - ParentEl If insert is wcTRUE and this value is non-zero, insert the new element into this container element.
wcBOOL - insert
  If insert is wcTRUE and ParentEl is non-zero insert the new element into ParentEl.
  If insert is wcTRUE and ParentEl is zero then insert the new element into the document.
  If insert is wcFALSE, the element is not inserted and must be inserted later by calling one of webC explicit element insert methods, the most common being wcDocInsertLast and wcElInsertLast.
int - top top position in pixels, absolute.
int - left left position in pixels, absolute.
int - height height in pixels or zero to not apply the height style and rely on defaults.
int - width width in pixels or zero to not apply the width style and rely on defaults.
int - zindex zindex value.
char - *color Named foreground (text) color or zero to use default, see wcElSetStyleColorNamed for available colors.
char - *bgcolor named background color or zero to use default.
char - *bgimage URL of background image (example: "file://images/mainpagebackground.gif") or , or zero for none.
char - *style Additional styles to apply or zero for none. Any style string may be provided for example" "visibilty:hidden;border-color:black;border-style:solid;border-thickness:2px;"
Returns:
wcEl the new element or zero if an error is detected or the element can't be created.
See Also:

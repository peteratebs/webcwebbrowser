wcELFind

Get the current input focus element for this document.

Insert an element into the first position of a document.
wcDocInsertLast

wcDocFindElement
Search for an element by id, name, type, and/or index
wcDocSetUrl

wcDocRefresh

wcDocClear

wcDocWriteHtml

wcDocWriteString

wcDocOpen
Open an HTML Document for writing
wcDocClose

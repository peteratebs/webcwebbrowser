void wcDocClose

(wcCtx* Ctx)

Close an HTML Document after writing content

Documentation

This function should be called on an HTML Document as soon as the user is finnished writing HTML and text content to it (wcDocWriteHtml, wcDocWriteString).

Parameters:
Ctx - The current wc context.
Returns:
0 if the operation was successful, -1 otherwise.
See Also:
HDOC_HANDLE, wcDocClear, wcDocOpen, wcDocWriteHtml, wcDocWriteString

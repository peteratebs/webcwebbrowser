void wcDocSetUrl

(wcCtx* Ctx, char* _url, wcBOOL refresh)

Set the URL of a document

Documentation

Sets the URL of the specified document. The new page will not be loaded (unless 'refresh' is set to WEBC_TRUE) until wcDocRefresh is called on this document.

Parameters:
Ctx - The current wc context.
url - The URL of the document to load
refresh - If TRUE, then load the given URL immediately and refresh the display; otherwise wait until wcDocRefresh is called.
See Also:
HDOC_HANDLE, wcDocRefresh

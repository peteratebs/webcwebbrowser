int wcWinLoadUrl

(wcCtx* Ctx, char* _url_str)

Load a location into the window and update the Ctx hdoc handle with new document information.

Documentation

This function clears the current content and reloads from another URL.

Parameters:
wcCtx - *Ctx Context of the window load into.
url_str - The URL to load
Returns:
0 on success, -1 otherwise

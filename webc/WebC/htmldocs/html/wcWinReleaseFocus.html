void wcWinReleaseFocus

(wcCtx* Ctx)

Release focus that was previously claimed for this browser view.

Documentation

Instructs webC to release the hold the browser view has on events dispatching and allow other browser views to recieve events.

Parameters:
wcCtx - *Ctx Context to claim focus for.
Returns:
Nothing
See Also:
wcWinClaimFocus

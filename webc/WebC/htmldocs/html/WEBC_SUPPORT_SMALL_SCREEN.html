C and HTML GUI for Embedded Platforms

#define WEBC_SUPPORT_SMALL_SCREEN

Set to 1 to for a reduced screen device.
Set to 1 to build for a reduced screen device. Defaults to smaller fonts. Use PDA chromebrowser by default, clip certain display elements instead of displaying horizontal scrollbar. 
         Note: use caution when enabling this option, it is an old option with some useful features but it has not been tested on recent releases of webC.

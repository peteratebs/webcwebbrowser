void wcImageGraphArc

(wcEL ImageElement, int ix, int iy, int ir1,
  int ir2, int ia1, int ia2, wcBOOL fill)

Draws an arc to an image canvas element.

Documentation

Draws an arc at the current pen width and color onto the background canvas element at coordinates provided by the caller.

Parameters:
wcEL - ImageElement - An image element that is the background canvas for our draw operations.
int - ix - x,y coordinate of the top left corner of the rectangle.
int - iy
int - ir1
int - ir2
int - ia1 - Start angle in degrees.
int - ia2 - End angle in degrees.
int - w - height and width of the ellipse.
int - h
wcBOOL - fill - Fill the area bounded by the arc if this argument is wcTRUE.
Returns:
nothing

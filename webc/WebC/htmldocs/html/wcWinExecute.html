int wcWinExecute

(wcCtx* Ctx)

Perform modal cycling of webC message loop on the context until stopped by wcWinExecuteStop.

Documentation

Cycles the webC message loop on the context until stopped by a call to wcWinExecuteStop from inside an event handler.

Parameters:
wcCtx - *Ctx Context of the window to cycle on.
Returns:
The integer return code passed to wcWinExecuteStop. Example:

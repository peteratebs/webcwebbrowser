HELEMENT_HANDLE webc_ElementGetLastChild

(HELEMENT_HANDLE element)

Get the last child of an element

Documentation

Returns the last child element of the given element.

Note: The wcEl support library provides a simplified DOM traversal package. Use of that interface is recommended for new projects.
Parameters:
element - The parent element
Returns:
The handle of the last child element or WEBC_NULL if no children.

HBROWSER_HANDLE webc_CreateBrowser

(struct HTMLBrowserConfig* config,
  WEBC_BOOL CreateWindow,
  char* _windowname,
  char* _home)

(Deprecated for most uses) - Create a new HTML Browser window

Documentation

Allocates resources for and displays a new web browser window with the specified home page document.
Deprecated for most uses, use wcWinCreateEx() instead. Called by wcWinCreateEx(), webc_CreateBrowser() .

Parameters:
config - The initial configuration of the new browser window
CreateWindow - If true call the GUI manager or frame buffer driver to create a new window.
windowname - Name of the window if one is being created.
home - URL of the initial page to load in this window. The browser will also return to this page when its 'Home' button (if it is configured to have one) is pressed, or when webc_BrowserHome is called.
Returns:
A handle to the newly created HTML Browser if successful, WEBC_NULL otherwise
See Also:
webc_DestroyBrowser, HTMLBrowserConfig

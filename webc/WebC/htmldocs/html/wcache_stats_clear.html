extern void wcache_stats_clear

(void)

Release all cached pages and images.

Documentation

An HTMLBrowser may allocate certain resources that are not released as soon as the browser is destroyed. This function is different from webc_Exit because webc_Init need not be called after calling it and further API calls.

Returns:
nothing

webc_Exit, webc_Init

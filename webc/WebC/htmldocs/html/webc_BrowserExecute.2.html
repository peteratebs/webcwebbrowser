int webc_BrowserExecute

(HBROWSER_HANDLE browser)

Process the input queue until this Browser is destroyed.

Documentation

This function will continuously process the input queue associated with the given browser until the browser has been destroyed via webc_DestroyBrowser

Parameters:
browser - The HTML Browser handle
Returns:
The value passed into webc_DestoryBrowser

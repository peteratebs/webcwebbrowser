wcEL wcDocGetFocus

(wcDOC Doc)

Get the current input focus element for this document.

Documentation

Get the handle to element that is the input focus for keyboard and mouse events for this document.
Parameters:
Doc - Handle of the document.
Returns:
Handle to the element, or 0.
See Also:
wcElSetFocus,wcWinSetFocus,wcWinGetFocus

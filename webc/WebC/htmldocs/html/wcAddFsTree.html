int wcAddFsTree

(const struct WebcVFileEntry* TableName)

Add a compiled image of a Windows subdirectory tree to webc's internal rom based file system.

Documentation

webC provides a handy rom based file system form managing content. Subdirectories on a Windows based PC can be compiled into C structure declarations by the tool makewebcfs and then inserted into the webC intoernal rom file system with this call. wcAddFsTree may be called multiple timesallowing software subsystems to individually register their content, independently of other modules.

Parameters:
const - struct WebcVFileEntry *TableName - The webc: file entry table name generated by makewebcfs.
Example:
You typed:
 makewebcfs -n=Examples \MyContent

This created the header files Examplesfiletable.h and Examplesfiledata.h.

Examplesfiledata.h contains the following declaration:

const struct WebcVFileEntry *Example[23] = { ...(if content had 22 files total..)

You edit a source file in your application startup sequence to include this line:

#include "Examplesfiletable.h"

Now add this call to your code so it executes before you try to access the files:

wcAddFsTree(Examples);

Returns:
0 or -1 on failure.

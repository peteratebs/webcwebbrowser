int webc_RegisterEventCallback

(HTMLEventCallback cb,
  char* argName)

Set a function to use as a custom event handler

Documentation

After this function is called, whenever an HTML tag loaded into WebC specifies its 'eventhandler' attribute (a WebC HTML extension) as the name passed in, all events that happen to that tag will be passed on to the C function specified as cb.

Parameters:
cb - The event handler callback.
name - The name of this event handler
Returns:
0 if the callback was successfully registered; -1 otherwise
See Also:
HELEMENT_HANDLE, SELECTED, webc_ElementGetValue

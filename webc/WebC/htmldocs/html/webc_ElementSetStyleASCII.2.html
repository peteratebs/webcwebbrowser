int webc_ElementSetStyleASCII

(HELEMENT_HANDLE element,
  char* _style, HTML_BOOL refresh)

Set style properties for an element

Documentation

This function will be operational once CSS support has been added to WebC.
Parameters:
element - The element to set style info for.
style - Cascading style sheet source data (e.g. "a.class1 { font-size: 12pt; }")
refresh - If WEBC_TRUE, then redraw this element

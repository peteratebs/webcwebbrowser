#define WEBC_SUPPORT_ERROR_PAGE_REDIRECT

Set to 1 to enable support for redirecting the URL when a URL load error occurs.
Set to 1 to enable loading of alternate URL when a load error occurs. Application specific content may be placed in url at the redirected page to provide application specific look and feel to the error messages and to invoke applications specific error handling code. 
		If WEBC_SUPPORT_ERROR_PAGE_REDIRECT is enabled you must call webc_setErrorRedirect() to assign redirect on error pages for
		URL_ERROR_SERVER_NOT_FOUND, URL_ERROR_PAGE_NOT_FOUND, URL_ERROR_PAYMENT_REQ and	URL_ERROR_FORBIDDEN.(examples are provided)

wcEL wcTableInsertRow

(wcCtx* Ctx, wcEL TableEl,
  CSSTextAlign align,
  CSSVerticalAlign valign, char* color,
  char* bgcolor, char* bgimage,
  char* style)

Insert a row into a table.

Documentation

Creates and inserts a row into a table.

Parameters:
wcCtx - * Ctx The current wc context.
wcEL - TableEl The table element returned from wcNewStyledTable
CSSTextAlign - align Horizontal alignment CSS_TEXT_ALIGN_CENTER,CSS_TEXT_ALIGN_JUSTIFY,CSS_TEXT_ALIGN_LEFT or CSS_TEXT_ALIGN_RIGHT
CSSVerticalAlign - valign Vertical alignment CSS_VERTICAL_ALIGN_BASELINE, CSS_VERTICAL_ALIGN_BOTTOM, CSS_VERTICAL_ALIGN_MIDDLE, CSS_VERTICAL_ALIGN_SUB, CSS_VERTICAL_ALIGN_SUPER, CSS_VERTICAL_ALIGN_TEXT_BOTTOM, CSS_VERTICAL_ALIGN_TEXT_TOP, CSS_VERTICAL_ALIGN_TOP
char - *color Named foreground (text) color or zero to use default, see wcElSetStyleColorNamed for available colors.
char - *bgcolor named background color or zero to use default.
char - *bgimage URL of background image (example: "file://images/mainpagebackground.gif") or , or zero for none.
char - *style Additional styles to apply or zero for none. Any style string may be provided for example" "visibilty:hidden;border-color:black;border-style:solid;border-thickness:2px;"
Returns:
wcEl the new element or zero if an error is detected or the element can't be created.

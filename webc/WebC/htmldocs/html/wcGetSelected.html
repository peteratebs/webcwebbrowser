int wcGetSelected

(HELEMENT_HANDLE element)

Get the selected option in a combo box

Documentation

Gets the index (beginning with 0) of the selected item in a combo box. The value (string) of the selected item of a combo box can be retrieved via wcElGetValue.
Parameters:
element - The combo box element to check.
Returns:
The selected index (0 - N), -1 on failure.

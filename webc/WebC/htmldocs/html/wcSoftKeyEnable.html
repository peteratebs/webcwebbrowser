void wcSoftKeyEnable

(HBROWSER_HANDLE hBrowser)

Enable webc's soft keyboard.

Documentation

This function should typically be called from startup code of touchscreen based applications to enable touchscreen support. On touch screen devices clicks on a text edit control pop up a softkeyboard control. This behavior must be enabled both at compile time by setting the constant WEBC_SUPPORT_TOUCHSCREEN in webcfg.h and at run time by calling wcSoftKeyEnable. This function is typically called once at startup on touchscreen based applications. On the desktop it is used for demonstrating touchscreen support where wcSoftKeyEnable enables the keyboard for a demonstration (ExSoftKeyPad)and then disables it when the example applet finishes.
Note: The softkeypad control is written in C++ using the webC 'C' so it should serve as a valuable example of a fairly complex application written with the webC API. The source code for the soft keyboard control resides in source/util/SoftKeyPad.cpp.

Parameters:
HBROWSER_HANDLE - hBrowser - An initialized browser handle that popup keyboard window control will create itself in.
Returns:
Nothing.

Appendix C

Using content imported from a file system in embedded webC applications. This section discusses how the "webc:" or "rom:" url can be used to organize your application and how to prototype content on Windows system and then import the content into the webC build image.
webcFsIntroduction
The webc: url (also accessible as rom:) is a rom based read only file system that enables you to include content in your application like you do from a file system.
makewebcfs
Compile content from a workstation subdirectory into a form suitable for use with webc's read onnly rom based file system.

void wcImageGraphDrawRect

(wcEL ImageElement, int x, int y, int w,
  int h, wcBOOL fill)

Draws a rectangle to an image canvas element.

Documentation

Draws a rectangle at the current pen width and color onto the background canvas element at coordinates provided by the caller.

Parameters:
wcEL - ImageElement - An image element that is the background canvas for our draw operations.
int - x - x,y coordinate of the top left corner of the rectangle.
int - y
int - w - height and width
int - h
wcBOOL - fill - Fill the area bounded by the rectangle if this argument is wcTRUE.
Returns:
nothing

const char* wcGetTypeString

(HTMLElementType TypeVal)

(not implemented) Return a pointer to an element type's string representation.

Documentation

Returns an ascii string containing the element's type or 0 if no match.
Example: const char *Type = wcElGetType(element) ? wcGetTypeString(wcElGetType(wcEL element)) : "Unknown type";
Parameters:
TypeVal - Type to retrieve string representation.
Returns:
An ascii string containing the element's type or 0 if no match.
See Also:
wcElGetType, HTMLElementType

webC browser API

Additional low level functions and unpublished API's (see webcsdkbrowser.cpp). APIs supporting specific event handlers by tag type, document URL history and system level function that are typically not called by applications code but are called by other API's and by the low level HID interface event handlers.
webc_BrowserAddToHistory
webC document history interface. Add a location to the HTML Browser history
webc_BrowserBack
Load the previous location in the history
webc_BrowserForward
Load the next location in the history
webc_BrowserHome
Reload the home page
webc_BrowserGetDocument
(Deprecated for most uses) Get an HTML browser's root document handle
webc_CreateBrowser
(Deprecated for most uses) - Create a new HTML Browser window
webc_DestroyBrowser
(Deprecated for most uses) - Close an HTML Browser window and free all its resources

wcEL wcWinGetFocus

(wcCtx* Ctx)

Search the context for element with focus.

Documentation

Finds the element in the document with focus.

Parameters:
wcCtx - *Ctx - Context to search.
Returns:
wcEL - The handle of the element with focus, 0 if none.
See Also:
wcWinSetFocus,wcWinGetFocusCtx,wcWinClaimFocus,wcWinReleaseFocus

#define WEBC_SUPPORT_INTERNAL

Set to 1 to enable support for WEBC: built in "rom" file protocol.
Set to 1 to enable the WEBC:// internal precompiled file lookup. The precompiled files must be built using the bintoc utility to create 'C' arrays from content source code which must then be manually inluded in webvfile.cpp. See appendix for instructions.

Vector graphics

Functions to perform simple line graphis, line, circle, ellipse, polygon, arc. rectangle.
wcImageGraphNameToColor
Convert a color index value to a 32 bit html value that may be used as a color argument to wcImageGraphSetPenColor.
wcImageGraphBeginDraw
Prepare an image for a sequence of one or more wcImageGraphDraw commands to follow.
wcImageGraphEndDraw
Close a sequence of draw commands and update the display.
wcImageGraphRefreshSrc
Reload the image canvas from source code.
wcImageGraphSetBrushWidth
Set the line width for subsequent draw operations.
wcImageGraphSetPenColor
Set the color for subsequent draw operations.
wcImageGraphDrawLine
Draw a line to an image element background canvas.
wcImageGraphDrawPolygon
Draw a sequence of lines to an image element background canvas.
wcImageGraphDrawRect
Draws a rectangle to an image canvas element.
wcImageGraphDrawEllipse
Draws an ellipse to an image canvas element.
wcImageGraphArc
Draws an arc to an image canvas element.
These functions perform vector graphics using an image as a canvas.

(void)

Set the color of an element using CSS.

Documentation

Formats a string and passes it to wcElSetStyle() to set the color a color of an element using CSS.

Parameters:
wcEL - Element - The element.
Prop - Which color property to set.
int - r (0-255).
int - g (0-255).
int - b (0-255).
refresh - Redraw the element immediately if wcTRUE. 

These are the most common properties that you might set.

CSS_PROPERTY_COLOR - The text color
CSS_PROPERTY_BACKGROUND_COLOR - The background

These properties may also be set.

CSS_PROPERTY_ALINK_COLOR,CSS_PROPERTY_VLINK_COLOR,CSS_PROPERTY_LINK_COLOR,CSS_PROPERTY_OUTLINE_COLOR,CSS_PROPERTY_BORDER_BOTTOM_COLOR,CSS_PROPERTY_BORDER_COLOR
CSS_PROPERTY_BORDER_LEFT_COLOR,CSS_PROPERTY_BORDER_RIGHT_COLOR,CSS_PROPERTY_BORDER_TOP_COLOR

Returns:
nothing.
See Also:
wcElSetStylePos,,wcElSetStyleColorHtml, wcElSetStyleColorNamed

int webc_HttpSetAuthorization

( HTTPAuthDomain* domain,
  const char* userName,
  const char* password,
  long ttlSec )

Set Authorization for HTTP site

Documentation

This method is used to set a username and password to access a protected HTTP server

Parameters:
domain - pointer to a struct decribing the authentication domain
userName - user name to set for this domain (string will be copied)
password - password to set for this domain (string will be copied)
ttlSec - time to live in seconds (0 for infinite) for user/password
See Also:
HTTPAuthDomain, webc_HttpClearAuthorization, webc_HttpSetAuthCallback, webc_HttpClearAllAuthorizations

void wcElSetStyleOpacity

(wcEL element, int Opacity,
  wcBOOL refresh)

Set the opacity of a div element.

Documentation

Formats a string and passes it to wcElSetStyle() to set the opacity a div element using CSS.

Parameters:
wcEL - Element - The element, must be a div
int - - opacity, must be between 0 and 100
refresh - Redraw the element immediately if wcTRUE.
Returns:
nothing.

wcElSetStyle

wcSys - System Oriented utility Functions

Timers, Virtual file support, memory management, string management, garbage collection, diagnostics.
wcCtxtInit
Initialize a webC context for use.
wcCtxtRelease
Garbage collect resources associated with a webC context.
wcAddFsTree
Add a compiled image of a Windows subdirectory tree to webc's internal rom based file system.
wcTimedCallback
Wait a number of miliseconds and then call the user callback function.
wcMemDbGetStats
Retrieve a compact summary of memory allocation statistics.
wcMemDbInstallPeakUsageCallback
Install a function to be called when peak memory allocation is reached.
wcMemDbGetReport
Returns a linked list of report records detailing memory usage.
wcMemDbGetWatchPoint
Returns a watchpoint list that may be used later to determine changes to the heap since this point.
wcMemDbReleaseReport
Free a memory report list that was returned from a wcMemDb function.
wcMemDbGetChangesSinceWatchPoint
Returns a list of changes to the heap since the watchpoint was established.
wcPrintMemStats
Calls wcMemDbGetStats and prints the results to the console.
wcPrintMemChangeReport
Print changes to the heap since the watchpoint was established.
wcPrintMemDetailReport
Specify sorting and summary options and print a heap report to the console.
These functions manage system level resource outside of the windows heirarchy.

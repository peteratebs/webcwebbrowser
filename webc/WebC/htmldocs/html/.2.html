voidwcJSIntroduction ()

(HDOC_HANDLE doc, const char* _source)

Call Javascript from native 'C' and execute the provided javascript source code

Documentation

This function allows a C function to called javascript to compile and execute source code. This function returns a handle to a script. wcJSDestroyScript() must be called to stop the script and free the script resources.

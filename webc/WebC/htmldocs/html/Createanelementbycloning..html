Create an element by cloning.

Documentation

Dynamically create a new element by cloning an existing element. If bCloneChildren is true, clone the element's children too. The element must be inserted into the document before much can be done with it. Note:

Parameters:
wcEL - El - The element to clone.
wcBOOL - bCloneChildren - wcTRUE or wcFALSE
Returns:
The new element or zero on failure.

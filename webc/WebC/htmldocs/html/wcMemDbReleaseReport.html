void wcMemDbReleaseReport

(RTPMemBlockHeader* pReport)

Free a memory report list that was returned from a wcMemDb function.

Documentation

Frees the memory returned from one of the functions that return a list. Non-zero values returned from wcMemDbGetWatchPoint, wcMemDbGetReport, wcMemDbGetChangesSinceWatchPoint must be freed by calling this function.

Parameters:
RTPMemBlockHeader - *pReport - The report list to free.

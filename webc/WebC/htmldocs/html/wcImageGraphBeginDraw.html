void wcImageGraphBeginDraw

(wcEL ImageElement)

Prepare an image for a sequence of one or more wcImageGraphDraw commands to follow.

Documentation

wcImageGraphBeginDraw clears the pallet to the orignial image it was created from so that a clean set of draw subroutines can be executed. The draw commands to follow will not be visible until wcImageGraphEndDraw is called.

Note: ImageElement is an image element that was either created with an html <div> tag or by a call wcNewStyledDiv or some other Javascript or C method that loads a valid image file.
Note: The following restrictions and limitations should be kept in mind.

  • The image content must exist, wcImageGraph does not support drawing to dynamically created elements with source not successfully set and loaded.
  • The image must be either 24 bit or 32 bit, not all image types and combinations are tested.
  • If you have any doubts try using a 24 or 32 bit .png image.
  • Position the image using its exact height and width. Do not scale the image using width and height properties or styles. Doing so will distort the pitch of drawn graphics.
  • Only operations sandwiched between the previous wcImageGraphBeginDraw, wcImageGraphEndDraw pair are visible.
Example: Create a pallet to draw into and draw crossing diagonal lines on it.


	wcEL ImageElement =  wcNewStyledImage(&childwindowCtx, 0, wcTRUE, "webc:///Example/Images/ExampleCanvas.png", 0, 0, 0, 200,200, 0, 0);
	wcImageGraphBeginDraw(ImageElement);
	wcImageGraphSetPenColor(ImageElement, wcImageGraphNameToColor(Blue));
	wcImageGraphDrawLine(ImageElement, 200, 200, 0, 0);
	wcImageGraphDrawLine(ImageElement, 0,  0, 200, 200);
	wcImageGraphEndDraw(ImageElement);

Parameters:
wcEL - ImageElement - An image element that is the background canvas for our draw operations.
Returns:
nothing

wcBOOL wcJSInitializeDocCtxt

(struct JSContext* cx, wcCtx* Ctx)

Initialize a webC context for use from a javascript context.

Documentation

This function should be called when entering a C handler from Javascript. It initializes a context block that may be passed to other wcAPI functions. When the C handler completes it should call call wcCtxtRelease before it returns. This will ensure that if any garbage was accumullated in the context block it will be freed.

Parameters:
cx - - The javascript context that was running
Ctx - - Pointer to a webC context structure to be initialized.
Returns:
wcTRUE if the javascript context could be mapped succesfully, otherwise wcFALSE.

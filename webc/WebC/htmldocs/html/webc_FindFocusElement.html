HELEMENT_HANDLE webc_FindFocusElement

(HBROWSER_HANDLE hBrowser)

Search for the element with focus.

Documentation

Does a depth-first search of the document tree of the document with focus for the element having focus.

Returns:
The handle of the element with focus. WEBC_NULL if none found.
See Also:
HELEMENT_HANDLE, webc_GetBrowserWithFocus()

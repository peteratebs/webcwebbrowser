void wcElSetInnerHtml

(wcEL El, char* newhtml)

Set an HTML element's inner html.

Documentation

Set an HTML element's inner html. Parses the provided Html source code and insert it into the the specified container element. The existing contents are freed and replaced with the elements created from the html text representation. 

 Examples:
     wcElSetInnerHtml(Elw, "<table border=\"1\">\<tr><td>(1,1)</td><td>(1,2)</td></tr></table>");

Parameters:
wcEL - Elw - The containing element.
char - *NewHtml - Html source code.
Returns:

wcWin - Window Oriented API Functions

Window Oriented API Functions
wcWinCreateEx
Create a browser window.
wcWinExecute
Perform modal cycling of webC message loop on the context until stopped by wcWinExecuteStop.
wcWinExecuteStop
Tell wcWinExecute to stop cycling and return a value to its caller.
wcWinLoadUrl
Load a location into the window and update the Ctx hdoc handle with new document information.
wcWinClaimFocus
Claim focus for a browser window.
wcWinReleaseFocus
Release focus that was previously claimed for this browser view.
wcWinSetFocus
Set the focus to a browser window and element.
wcWinMoveTo
Move a browser window.
wcWinBringToTop
Raise window to z-order stack top.
wcWinGetPos
Get the displayed position of a window.
wcWinSetPos
Set the position, height and width of a window.
wcWinGetFocusCtx
Return the a Context initialized for the window with focus..
wcWinGetFocus
Search the context for element with focus.
wcWinClose
Close a browser window.
These functions create and manage individual windows containing browser instances.

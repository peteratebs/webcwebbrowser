void wcElPopEventHandler

(wcEL element)

Restore the previous event handler for this element.

Documentation

Pop the previous event handler for this element from an internal stack and make it the new event handler for the specified element. This function returns the event handler to the handler prior to the last call to wcElSetEventHandler. This push pop capability simplifies implementing effects like drag and drop, where a new event handlers are assigned when dragging starts, and released when it stops.

Note: See wcElSetEventHandler.

Parameters:
wcEL - Element - The element. 
Example:
See the example for wcElSetEventHandler.
case HTML_EVENT_CLICK            :
// Revert to previous handler when clicked.
wcElPopEventHandler(element);
return (HTML_EVENT_STATUS_HALT);

Returns:
Nothing

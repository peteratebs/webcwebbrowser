wcEL wcWinGetFocus

(wcCtx* Ctx)

Search for the element and or window with focus.

Documentation

If the Context is uninitialized find the browser window with focus. Do a depth-first search of the document tree of the focus window for the element having focus.

Returns:
wcEL - The handle of the element with focus.
See Also:
wcWinSetFocus,wcWinClaimFocus,wcWinReleaseFocus

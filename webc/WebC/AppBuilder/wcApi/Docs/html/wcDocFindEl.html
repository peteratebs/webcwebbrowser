wcEL wcDocFindEl

(wcCtx* Ctx, char* id)

Search the document for a specific id.

Documentation

Search of the document tree for the first element with the specified id.

Parameters:
wcCtx - * Ctx - The current wc context.
char - * Id - The id to search for.
Returns:
The handle of the element with the specified id, or zero.

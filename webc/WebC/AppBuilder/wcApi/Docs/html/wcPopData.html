void wcPopData

(wcCtx* Ctx, wcEL Element)

Pop the current data pointer from the a window or element within the window.

Documentation

Parameters:
wcCtx - * Ctx - Context to retrieve private data from.
wcEL - Element - Element whose data stack is to be popped. If this value is zero the context data stack is popped.
Returns:
nothing

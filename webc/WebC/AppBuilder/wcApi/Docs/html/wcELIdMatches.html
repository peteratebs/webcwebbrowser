wcBOOL wcELIdMatches

(wcEL element, char* string)

Compare an element's id to a string.

Documentation

Compare an element's id to a string, return true if they match.

Parameters:
wcEL - El - The element to compare.
char - * IdStr - The string to match.
Returns:
wcTRUE if the element has an assigned id and they match wcFALSE otherwise.

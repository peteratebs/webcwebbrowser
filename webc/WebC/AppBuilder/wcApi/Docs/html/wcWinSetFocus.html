wcBOOL wcWinSetFocus

(wcCtx* Ctx, wcEL Element)

Set the focus to a browser window and element.

Documentation

Instructs webC to route keyboard events to the current window.

Parameters:
wcCtx - *Ctx - Window to set focus to. Current context if window already has focus.
wcEL - Element - Element in the window. If zero use the windows current focus element.
Returns:
wcTRUE on success.
See Also:
wcWinGetFocus

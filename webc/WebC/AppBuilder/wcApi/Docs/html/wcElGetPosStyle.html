wcBOOL wcElGetPosStyle

(wcEL element, int* top, int* left,
  int* width, int* height)

Get the style positioning values of an element.

Documentation

Get the current position of an element from its CSS or HTML positioning. Fails if the element has not been positioned. Note: wcElGetPosStyle can only obtain the coordinates of elements that were manually sized and positioned with CSS or HTML attributes. The routine can be used in screen constructors on elements that have been positioned but not yet drawn. After an is drawn, whether manually positioned or not, coordinates may be obtained by calling wcElGetPosWin. Note: top, left height and width are returned as zero if they have not been set.

Parameters:
wcEL - Element - The element.
int - * top - top goes here. If NULL no result is returned for top.
int - * left - left goes here. If NULL no result is returned for left.
int - * height - height goes here. If NULL no result is returned for height.
int - * width - width goes here. If NULL no result is returned for width.
Returns:
wcTRUE if the argument is valid.
See Also:
wcElGetPosStyle,wcElSetPosStyle

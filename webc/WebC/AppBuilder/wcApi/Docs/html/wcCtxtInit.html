void wcCtxtInit

(wcCtx* Ctx, wcBROW browser, wcDOC doc)

Initialize a webC context for use.

Documentation

Initialize a webC context for use. This function must be called by event handlers to create a browser context for passing to wcAPI functions.

Parameters:
wcCtx - * Ctx - The wc context to initialize.
wcBROW - Browser - The browser.
wcDOC - Document - The document.
Returns:
nothing
